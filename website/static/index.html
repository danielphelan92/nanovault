Compatibility

Which iPods work?

If Windows shows your iPod as a drive, NanoVault can read it. That covers the classic disk-mode iPods.

Works

  • iPod nano (4th gen) — the main target
  • iPod nano 1st–5th gen
  • iPod classic / video / photo
  • iPod mini

Best effort

  • iPod shuffle — songs copy fine; some names may fall back to the filename
  • Older / unusual databases — always falls back to reading the songs directly
  • Any iPod with “Enable disk use” switched on

Not supported

  • iPod touch & iPhone — they don’t mount as a drive
  • iPod nano 6th / 7th gen — no disk mode

Frequently asked questions

Is it really free? What’s the catch?
Yes — every feature, no trial, no track limit, no “pro” tier. It’s open source under the MIT licence. If you want to support it, there’s a coffee button; that’s the only ask.
Will it change or wipe my iPod?
No. NanoVault opens your iPod strictly read-only — it’s designed so it literally can’t write to the device. It only ever copies music off it.
Do I need iTunes?
No. As long as Windows shows your iPod as a drive, NanoVault works on its own. On some older models you may need to switch on “Enable disk use” once — the app walks you through it.
Windows warned me about the download. Is it safe?
The installer isn’t code-signed yet, so Windows SmartScreen may show a “more info” prompt on first run — that’s normal for small free apps. The full source is on GitHub if you’d like to build it yourself instead.
What about protected (purchased) songs?
They’re copied exactly as they are and clearly labelled. NanoVault never removes DRM — playing protected files still needs the Apple account they were bought with.
Which iPods are supported?
Classic disk-mode iPods — nano 1st–5th gen (4th gen is the main target), iPod classic, video, photo and mini. iPod touch/iPhone and the nano 6th/7th gen don’t mount as drives, so they aren’t supported. See the compatibility list.
